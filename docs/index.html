getOnPath

Return's value on given path

getOnPath
Parameters
object (object)
path (array)
Returns
any: value on given path or undefined if invalid path
Example
getOnPath({foo: {bar: 'baz'}}, ['foo', 'bar']}
// 'baz'
getOnPath({foo: {bar: 'baz'}}, ['foo']}
// {bar: 'baz'}
getOnPath({foo: {bar: 'baz'}}, ['baz']}
// undefined

mergeDeep

Merge two objects deeply together

Objects and their values of type objects are merged except Arrays. Arrays are treated as scalar

mergeDeep
Parameters
currentObject (object)
values (object)
Returns
object: Result object
Example
mergeDeep({scalar: 1, foo: 'foo'}, {scalar: 2, bar: 'bar'}
// {scalar: 2, foo: 'foo', bar: 'bar'}
mergeDeep({a: [1,2,3]}, {a: [4,5,6]}
//  {a: [4,5,6]}
mergeDeep({f: () => 'foo'}, {f: () => 'bar'}
//  {f: () => 'bar'}
